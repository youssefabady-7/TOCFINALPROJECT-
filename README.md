# TOC Final Project

This repository contains two independent Java console projects for Formal Languages and Automata Theory.

## Project 1: RegexToDFAProject

Regular Expression to DFA Conversion.

Main features:
- Enter custom regular expressions.
- Validate regex syntax.
- Convert regex to NFA using Thompson's Construction.
- Convert NFA to DFA using Subset Construction.
- Minimize the DFA.
- Display states and transitions.
- Test custom strings.
- Run a built-in professor demo test case.

## Project 2: NFAToDFAVisualSimulator

NFA to DFA Visual Simulator.

Main features:
- Enter custom NFA states, alphabet, start state, final states, and transitions.
- Convert NFA to DFA.
- Display NFA and DFA transition tables.
- Test custom strings.
- Run a built-in professor demo NFA.

## Notes

Each project has its own:
- `README.md`
- `TEST_CASES.txt`
- PDF explanation file
- `src` folder

The projects are intentionally separate. Do not compile both `src` folders together because they use some of the same class names independently.
